Исходный код сайта deadchannel.ru, содержащего афишу предстоящих
околоиндустриальных мероприятий Санкт-Петербурга.

Главная функция сайта — отправка напоминаний о приближающихся событиях по
электронной почте и SMS.  Напоминания отправляются за неделю и за сутки до
события.  Для отправки SMS используется API сайта sms.ru.

Настройки хранятся в файле config.py, в качестве образца можно использовать
config.py.example.
